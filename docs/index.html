F O R M U L A T E

A RuffLIB library for transforming
an object schema into an HTML form.


Version: 0.0.1
Repo: https://github.com/richplastow/rufflib-formulate
Homepage: https://richplastow.com/rufflib-formulate
Tests: run-browser-tests.html
Demo 1: demo/demo-1.html
Typical usage:
new Formulate(
    document.querySelector('#wrap'),
    'my_form',
    {
        _meta: { title:'My Form' },
        outer: {
            _meta: { title:'Outer' },
            an_inner_boolean: Formulate.boolean(false),
            another_boolean: Formulate.boolean(true),
        },
        outer_boolean: Formulate.boolean(true),
    }
);
